# Routing в React приложениях c использованием react-router

Приложение для демонстрации.
Состоит из нескольких страниц: главная, список пользователей, информация о кокретном пользователе, список плейлистов, информация о конкретном плейлисте.
Функционал: навигация с отображением в адресной строке, фильтрация списка по двум параметрам через поисковые запросы (при копировании и открытии URL текущее положение и результаты фильтрации сохраняются)

В качестве сборщика используется [vite](https://vitejs.dev/)
